# K-Means Clustering — Task 13

## AI & ML Internship

### About
Customer segmentation using K-Means Clustering
on Mall Customers Dataset with Elbow Method
for optimal cluster selection.

---

### Dataset
- Name     : Mall Customers Dataset
- Rows     : 200
- Features : Gender, Age, Annual Income, Spending Score

---

### Steps Completed
1. Created and explored Dataset
2. Feature Scaling using StandardScaler
3. Elbow Method for Optimal K
4. K-Means Model Training
5. Cluster and Centroid Visualization
6. Silhouette Score Calculation
7. Silhouette Score Comparison across K values
8. Cluster Profile Summary

---

### Results
- Optimal K        : 5
- Silhouette Score : 0.55-0.65 (Good clustering)
- Segments Identified : Target, Careful, Standard,
                         Sensible, Careless Customers
